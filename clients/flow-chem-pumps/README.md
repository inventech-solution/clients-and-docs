## 4. Core Business Challenge

**SEO is not delivering results.**

The website has virtually no organic visibility — only ~5,000 impressions/month on Google Search Console with most target keywords buried on Page 3+. This is a multi-layered problem:

- Technical SEO issues on a Custom CMS
- Thin and low-quality content across product/category pages
- No backlink profile worth speaking of
- Poor on-page optimisation (titles, meta, headings, internal linking)

For a B2B brand where purchase decisions start with a Google search, this is a critical gap. The organic channel is essentially non-functional right now.
