## 2. What They Sell

Docuforte is a **document management, digitisation, and records intelligence platform** — an end-to-end solution covering physical-to-digital conversion, cloud-based document management, AI-powered search and retrieval, and secure records lifecycle management.

### Platform Features

| Feature | Description |
|---|---|
| **AI-Powered Search** | Search across content, metadata, and handwritten notes in scanned files |
| **AI Chatbot** | Natural-language queries answered directly from document content |
| **Online Document Editing** | Edit Word, Excel, and PDF in-browser with auto-save and versioning — no local software needed |
| **Multi-Tenant Architecture** | Manage multiple organisations, departments, and users with clean data separation |
| **Secure File Sharing** | Share via QR codes or email with password protection and granular access rights |
| **Email-to-Dashboard** | Auto-ingest incoming emails and attachments with predefined routing rules and tags |
| **Versioning & Audit Trails** | Full action history — who changed what and when |
| **Physical File Mapping** | Connect digital records to physical counterparts with movement tracking and receipt alerts |
| **Security & Compliance** | Role-based access control, data masking, watermarking, protected marks |
| **Sustainable Infrastructure** | Hosted on AWS powered by 100% renewable energy — supports paperless + carbon-efficient ops |

### Services

| Service | Description |
|---|---|
| **Scanning & Digitisation** | High-volume conversion of physical documents (invoices, HR files, legal dockets) into PDF/A or TIFF |
| **Offsite Records Storage** | Secure physical archive storage with pickup, sealed manifests, and barcode inventory tracking |
| **Secure Shredding** | Auditable record destruction with Certificate of Destruction |
| **Document Hosting** | Secure cloud-based or on-premises content hosting |
| **Virtual Data Room (VDR)** | Secure environment for sensitive data sharing during transactions or audits |

**Positioning:** Not just a DMS — Docuforte is a full records intelligence platform covering the entire document lifecycle: digitise → manage → search → share → store → destroy. The AI layer (search + chatbot) is a strong differentiator in the India market.
